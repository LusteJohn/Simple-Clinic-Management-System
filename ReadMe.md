# Simple Clinic Appointment and Queue Management System

A simple and modern web-based **Clinic Appointment and Queue Management System** designed to simplify appointment scheduling and patient queue management for small to medium-sized clinics. The system enables patients to book appointments online while allowing clinic staff to efficiently organize schedules, monitor daily queues, and manage patient records through an intuitive dashboard.

Built with **Native PHP** for the backend and **Vue.js** for the frontend, the application combines a lightweight server-side architecture with a responsive and interactive user interface. The system focuses on improving clinic workflow by reducing manual appointment handling, minimizing patient waiting times, and providing real-time queue monitoring.

## Features

* Secure user authentication
* Patient appointment booking
* Appointment approval and management
* Real-time queue monitoring
* Queue number generation
* Patient record management
* Appointment history tracking
* Search and filtering functionality
* Responsive dashboard for clinic staff
* Clean and user-friendly interface

## Tech Stack

* **Frontend:** Vue.js
* **Backend:** Native PHP
* **Database:** MySQL
* **Development Server:** Herd
* **Languages:** HTML, CSS, JavaScript, PHP

## Project Goals

This project was developed to strengthen full-stack web development skills by combining Native PHP with Vue.js to build a simple but practical healthcare application. It demonstrates concepts such as authentication, CRUD operations, appointment scheduling, queue management, database relationships, and responsive user interface development. The project also serves as a foundation for future enhancements such as email notifications, SMS reminders, online consultations, and analytics.
